
##ReplicationCheckerUtility
## Created by: Ahmed Lotfy
## Description:
a utility used to check any Replication Failure Prompt between all DCs in any Domain Environment.
## Guide:

1- Edit the Smtp Server, Mail Sender, mailto Variables to your Desired values.
2- Ensure that you are runing the script as an administrator.
3- Ensure that your CAS Role or any Client access Equivalent in your mailing server is Configured to receive emails from Mail Sender address.
